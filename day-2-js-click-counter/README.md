# Day 2 – JavaScript Click Counter

This project is a simple click counter built with HTML and JavaScript.

## What I learned
- JavaScript variables (`let` and `const`)
- DOM manipulation
- Event listeners (`addEventListener`)
- Updating HTML content dynamically

## How it works
Each time the button is clicked, the counter increases by one and the text on the page updates.

## How to run
1. Open `index.html` in a browser
2. Click the button
3. Watch the counter increase
